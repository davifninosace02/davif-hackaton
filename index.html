Guía Rápida de Inicio

# 1. Clona el repositorio
git clone https://github.com/ninos/contrarian-coder

# 2. Instala dependencias
npm install

# 3. Inicia el modo desarrollo
Presiona F5 (Debug Mode)

# 4. Busca el panel
En la nueva ventana, busca 'Contrarian Coder' en el sidebar / le ha el README.md

⚙️ Configuración (API Keys)

Para activar la magia, pega tu API Key en la sección de ajustes del panel. Soporta OpenRouter y Gemini. Si no tienes una, usará un modelo gratuito por defecto.
