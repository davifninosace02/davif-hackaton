🦾 Contrarian Coder

Extensión para Visual Studio Code que desafía tu código y te ayuda a mejorarlo, proporcionando alternativas de alto nivel y mejoras en tiempo real.

¿Qué es Contrarian Coder?

Contrarian Coder es una extensión para Visual Studio Code diseñada para ser tu crítico de código. En lugar de confirmar que todo está bien, la extensión identifica posibles fallos y te ofrece alternativas de mayor calidad para mejorar tu código. ¡Es como tener un asistente que te desafía a ser mejor!

✨ Características Principales

  • 🧩 **Caminos Proactivos**: Obtén dos enfoques distintos (Camino 1 y Camino 2) para resolver un problema de código.
  • ⚡ **Aplicar Código**: Con un clic, puedes aplicar directamente las sugerencias de código que la extensión te ofrece.
  • 🛡️ **Soporte Multi-IA**: Funciona con OpenRouter (modelos gratuitos) y Gemini, proporcionando diversas opciones para sugerencias de IA.
  • 🔍 **Detección de Errores y Patrones**: Contrarian Coder analiza tu código buscando errores comunes y patrones de mejora.
